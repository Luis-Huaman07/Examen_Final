# Sistema de Reservas para una Cafetería Universitaria  
Proyecto en C# con uso de matrices, funciones propias y manejo de cadenas.

---

##  Descripción del problema
Una cafetería universitaria recibe reservas de combos por turnos (mañana y tarde).  
Cada turno puede registrar hasta **20 reservas**.  
Cada reserva contiene:
- Nombre del estudiante
- Combo elegido (nombre)
- Precio del combo

El sistema debe permitir registrar, cancelar, listar y buscar reservas, además de calcular ingresos.

---

###  Menú inicial de combos
Al iniciar el programa, el usuario define **mínimo 3 combos**, cada uno con nombre y precio.
